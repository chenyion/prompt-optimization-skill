# Prompt 工程自动化调优 Skill

## Overview
针对大模型应用场景，自动分析现有 Prompt 的有效性、冗余度、歧义点，基于目标优化结构、少样本示例、思维链引导，生成多组备选并执行 A/B 测试，输出最优版本与调优报告。

## Use Cases
- 当你需要优化现有 Prompt 以提升生成效果时
- 当你需要对多组 Prompt 进行并行效果对比时
- 当你需要沉淀场景化 Prompt 模板时

## Workflow
1. **输入分析**：接收原始 Prompt 和场景目标（如“代码生成准确率”“文案相关性”）
2. **自动诊断**：分析冗余度、歧义点、指令清晰度，输出诊断报告
3. **Prompt 优化**：基于诊断结果，自动生成 3-5 组优化版本（含结构调整、少样本补充、思维链引导）
4. **A/B 测试**：并行调用目标大模型，对比准确率、相关性、稳定性
5. **结果输出**：返回最优 Prompt、调优报告，支持沉淀至模板库

## Examples
### 示例 1：代码生成 Prompt 优化
> 用户输入：“帮我写个 Python 排序函数”  
> 优化后输出（含思维链引导）：  
> “请编写一个 Python 快速排序函数，要求：1. 支持自定义比较函数；2. 包含时间复杂度注释；3. 先思考排序逻辑，再逐步实现代码。”

### 示例 2：A/B 测试流程
> 输入：原始 Prompt + 3 组优化版本 + 测试用例（10 条代码生成需求）  
> 输出：各版本的准确率（85% vs 92% vs 90%）、最优版本推荐

## Technical Details
- 依赖：需配置目标大模型 API（如 GPT-4o、Claude 3.5）
- 权限：网络访问（调用大模型）、文件读写（保存报告和模板）
- 评估指标：准确率（人工标注/规则匹配）、相关性（余弦相似度）、稳定性（多次生成结果方差）

## Notes
- 若未指定场景目标，默认优化“通用指令清晰度”
- A/B 测试支持自定义测试用例，或自动生成场景相关测试数据
